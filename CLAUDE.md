# Squadron — Claude Code

> Lire **[AGENTS.md](AGENTS.md)** en premier. Toute la documentation nécessaire pour travailler efficacement sur ce projet s'y trouve.

## Démarrage

```bash
cd server && npm start                   # serveur → port 3001
cd client && python -m http.server 8080  # client  → http://localhost:8080/Sprite%20Forge.html
```

## Règle fondamentale

Après tout changement significatif (nouveau fichier, endpoint, convention de rendu, logique de combat), **mettre à jour AGENTS.md** ou le sous-doc concerné dans `docs/`.
